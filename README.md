# 🌱 PlantCare

PlantCare is a responsive frontend web application that helps users explore indoor plants, learn proper plant care, and maintain a personalized Care List. The project is built using HTML, CSS, Bootstrap, and JavaScript with Local Storage for authentication and data persistence.

---

## 📌 Features

- 🌿 User Signup & Login
- 🔐 Authentication using Local Storage
- 🌱 Explore Different Plants
- 🔍 Search Plants
- 📂 Category Filter
- 📖 Plant Care Guide (Bootstrap Modal)
- ❤️ Personal Care List
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 🚀 Smooth UI with Bootstrap 5

---
